# Media Kit

Use this document as a short public description source for SharpMTKClient.

## Short Description

SharpMTKClient is a professional C# / .NET source code project for MediaTek / MTK protocol development on Windows.

## Long Description

SharpMTKClient is a developer-focused source code project for building Windows-based MediaTek / MTK service software. It includes development paths for device initialization, BROM / Preloader communication, Download Agent workflows, flash operations, partition management, file-system access, NV / IMEI logic, RPMB workflows and selected security research-oriented modules.

The public repository is a product showcase only. The full source code package is available through private licensing.
